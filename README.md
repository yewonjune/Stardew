# 🌱 Night of Petals (Unity)
## 실행화면
Unity 2D 기반 스타듀밸리 스타일 농장 & 생활 시뮬레이션 게임
농사, 전투, NPC 상호작용, 컷신, 세이브/로드까지 실제 상용 게임 구조를 목표로 개발한 개인 프로젝트입니다.

## 📌프로젝트 개요

개발 기간: (예: 2025.03 ~ 2025.12)
개발 인원: 1인 개발
플랫폼: PC (Windows)

## 🛠️ 사용 기술
Unity 2D / PhotoShop / AI
Firebase(Realtime Database)
DOTween / Odin Inspector

## 🎮 주요 콘텐츠
✔️ 기본 플레이 루프
하루 단위 시간 흐름 (Day / Hour / Minute)
행동에 따른 스태미나·체력 소모
하루 종료 → 저장 → 다음 날 시작

✔️ 농장 시스템
Tilemap 기반 토지 관리
경작 / 물주기 / 작물 성장
계절 변화에 따른 맵 및 환경 변화

✔️ NPC 시스템
NPC 스케줄 이동 (시간 기반)
대화 시스템 (Dialogue Data 기반)
호감도 시스템 및 상태 저장
컷신 트리거 및 연출

✔️ 전투 & 광산
층별 광산 스테이지 구조
광물·자원 스폰 시스템
보물방(특수 층) 처리

✔️ UI 시스템
인벤토리
퀘스트 게시판
랭킹 UI
